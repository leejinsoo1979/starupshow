# 01. 개요 및 철학

## 📖 문서 목적

본 문서는 **GlowUS 플랫폼의 NeuraMap 앱**에 통합할 AI 코딩 에이전트 개발의 완전한 명세서입니다. Cursor, AntiGravity, Claude Code 등 기존 코딩 에이전트의 핵심 패턴을 분석하고, TypeScript+React 프론트엔드와 Node.js/Python 백엔드, 다중 LLM API를 활용한 프로덕션급 구현 가이드를 제공합니다.

---

## 🎯 프로젝트 목표

### 핵심 미션
**"개발자가 자연어로 지시하면, AI가 자율적으로 코드를 작성·테스트·커밋하는 완전 자동화된 개발 워크플로우 구축"**

### 구체적 목표

1. **자율성 (Autonomy)**
   - 사용자의 요구사항을 이해하고 독립적으로 작업 계획 수립
   - 다단계 작업을 에이전트가 스스로 분해하고 실행
   - 에러 발생 시 자가 진단 및 수정 시도

2. **신뢰성 (Reliability)**
   - 모든 변경 사항은 자동 테스트로 검증
   - 사용자 승인 게이트를 통한 안전장치
   - 실패 시 자동 롤백 메커니즘

3. **투명성 (Transparency)**
   - 모든 의사결정 과정을 사용자에게 설명
   - 계획 단계에서 작업 목록을 명시적으로 제시
   - 변경 영향도를 사전에 보고

4. **확장성 (Scalability)**
   - 소규모 버그 픽스부터 대규모 리팩토링까지 대응
   - 멀티 파일, 멀티 모듈 작업 지원
   - 코드베이스 크기에 무관하게 작동

---

## 🧠 핵심 설계 철학

### 1. Plan-First Approach
> **"생각하지 않고 코드를 짜지 마라"**

- 모든 작업은 **명시적 계획(Plan)**으로 시작
- 계획은 사용자가 읽고 검토할 수 있는 형태로 제시
- 계획 승인 후에만 실행(Execute) 단계로 진입

### 2. Test-Driven Verification
> **"테스트하지 않은 코드는 존재하지 않는다"**

- 모든 코드 변경 후 자동으로 빌드/린트/테스트 실행
- 테스트 실패 시 자동으로 수정 재시도 또는 사용자에게 보고
- 회귀 테스트로 기존 기능 보호

### 3. Human-in-the-Loop
> **"AI는 제안하고, 인간이 승인한다"**

- 중요한 의사결정 지점마다 사용자 승인 필요
- 사용자는 언제든 개입하여 방향 수정 가능
- AI 출력물은 항상 사람의 최종 검토 필요

### 4. Context-Aware Intelligence
> **"코드베이스를 이해하지 못하면 수정할 수 없다"**

- 코드 지식 그래프(Code Knowledge Graph)로 구조 파악
- LSP(Language Server Protocol)로 심볼 관계 추적
- 변경 영향도 분석으로 리스크 최소화

### 5. Memory-Driven Evolution
> **"과거를 기억하지 못하면 미래를 계획할 수 없다"**

- Git-Context-Controller(GCC)로 에이전트 메모리 버전 관리
- plan.md 파일에 작업 이력 지속적 기록
- 세션 간 컨텍스트 공유로 장기 프로젝트 대응

## 🔍 참조 시스템 분석

### Cursor

**핵심 특징:**
- IDE 통합형 AI 코딩 어시스턴트
- 실시간 코드 제안 및 자동완성
- 채팅 기반 코드 생성

**학습 포인트:**
- 에디터와 긴밀한 통합
- 컨텍스트 인식 자동완성
- 멀티 파일 편집 지원

**차별화 전략:**
- Cursor는 제안 중심 → NeuraMap은 **자율 실행** 중심
- Cursor는 Claude만 지원 → NeuraMap은 **멀티 모델** 지원

### Claude Code (CLI)

**핵심 특징:**
- 터미널 기반 코딩 에이전트
- Read, Write, Edit, Bash 등 파일 시스템 도구 제공
- Streaming 응답으로 실시간 피드백

**학습 포인트:**
- Tool Calling 시스템 설계
- 파일 시스템 직접 조작
- Bash 명령 실행 능력

**차별화 전략:**
- Claude Code는 CLI 전용 → NeuraMap은 **GUI + 워크플로우 빌더**
- Claude Code는 단일 세션 → NeuraMap은 **GCC로 장기 메모리**

### AntiGravity

**핵심 특징:**
- AI 기반 프로젝트 스캐폴딩
- 전체 프로젝트 구조 생성
- 프레임워크별 베스트 프랙티스 적용

**학습 포인트:**
- 대규모 코드 생성 능력
- 프레임워크 템플릿 관리
- 프로젝트 초기화 자동화

**차별화 전략:**
- AntiGravity는 초기 생성 전문 → NeuraMap은 **유지보수 + 진화** 전문
- AntiGravity는 단발성 → NeuraMap은 **지속적 개선 루프**

## 💎 핵심 가치 제안

### 개발자에게
- **시간 절약**: 반복적인 코딩 작업 자동화
- **품질 향상**: 자동 테스트로 버그 사전 차단
- **학습 도구**: AI의 코드를 보며 베스트 프랙티스 습득

### 팀에게
- **일관성**: 코딩 스타일과 패턴의 자동 적용
- **문서화**: 모든 변경 사항이 명확한 커밋 메시지로 기록
- **지식 공유**: 에이전트 메모리로 팀 지식 축적

### 조직에게
- **생산성**: 개발 속도 2~3배 향상 목표
- **안정성**: 자동 테스트로 프로덕션 버그 감소
- **확장성**: 신규 개발자 온보딩 시간 단축

## 📊 성공 지표 (KPI)

### 정량적 지표
1. **작업 완료율**: 사용자 요청 중 성공적으로 완료된 비율 (목표: 80%+)
2. **테스트 통과율**: 에이전트가 생성한 코드의 테스트 통과 비율 (목표: 95%+)
3. **재시도율**: 실패 후 재시도가 필요한 비율 (목표: <20%)
4. **평균 작업 시간**: 요청부터 커밋까지 소요 시간 (목표: <10분)

### 정성적 지표
1. **사용자 만족도**: 개발자 피드백 점수 (목표: 4.5/5)
2. **코드 품질**: 동료 리뷰에서의 코드 품질 평가
3. **신뢰도**: 에이전트 제안을 그대로 승인하는 비율

## 🚀 프로젝트 범위

### Phase 1: Core Engine (v0.1)
- Basic Agentic Loop (Plan → Modify → Verify → Commit)
- 기본 Tool API (read, write, edit, run)
- 단일 파일 수정 지원

### Phase 2: Advanced Tools (v0.2)
- Code Knowledge Graph
- LSP 통합 (symbols, diagnostics)
- 멀티 파일 리팩토링

### Phase 3: Memory & Learning (v0.3)
- Git-Context-Controller (GCC)
- plan.md 자동 생성
- 세션 간 메모리 공유

### Phase 4: Enterprise Features (v1.0)
- 팀 협업 기능
- 커스텀 룰셋
- 성능 최적화
