## 1. 혁신적 스킬 (Creative/Imaginative)

### 1.1 인지 확장 스킬

#### "만약에" 시뮬레이터
- **입력**: 비즈니스 가정 질문 (예: "만약 우리가 가격을 30% 올리면?")
- **출력**:
  - 고객 반응 예측
  - 매출 변화 시뮬레이션
  - 경쟁사 대응 시나리오
- **가치**: 의사결정 전 리스크 사전 파악

#### 실패 예언자
- **입력**: 프로젝트 계획서/아이디어
- **출력**: "이 프로젝트가 망하는 10가지 시나리오"
- **가치**: 시작 전 방어 전략 수립, 블라인드 스팟 발견

#### 아이디어 충돌기
- **입력**: 2개의 전혀 관련 없는 산업/개념
- **출력**: 융합 혁신 아이디어 리스트
- **예시**: "치과 + 넷플릭스 = 구독형 치아 관리 서비스"
- **가치**: 창의적 발상, 블루오션 발굴

---

### 1.2 디지털 분신 스킬

#### 나 대신 답장
- **입력**: 사용자의 과거 이메일/슬랙 히스토리 학습
- **기능**:
  - 내 말투로 답장 초안 생성
  - 톤 조절 (공격적/부드럽게 등)
- **가치**: 커뮤니케이션 시간 단축, 일관된 톤 유지

#### 미래의 나에게 질문
- **입력**: 현재 프로젝트/업무 데이터
- **기능**: "6개월 후의 나"를 시뮬레이션
- **출력**: 미래 관점에서의 조언
- **가치**: 장기적 관점 확보, 후회 방지

#### 과거의 나 소환
- **입력**: 과거 프로젝트 문서들
- **기능**: 과거 의사결정의 맥락 복원
- **출력**: "그때 왜 이런 결정을 했는지" 설명
- **가치**: 조직 지식 보존, 컨텍스트 복구

---

### 1.3 예지 스킬

#### 트렌드 레이더
- **입력**: 특정 키워드/기술/산업
- **기능**: 글로벌 트렌드 실시간 추적
- **출력**: "6개월 후 뜰까 죽을까" 예측 리포트
- **가치**: 선제적 투자/철수 결정

#### 숨은 기회 탐지기
- **입력**: 관심 산업/분야
- **분석 소스**: 뉴스 + 특허 + 논문 + SNS 크로스 분석
- **출력**: "아무도 안 보는데 곧 터질 것들" 리스트
- **가치**: 경쟁사보다 먼저 기회 포착

#### 경쟁사 다음 수 예측
- **입력**: 경쟁사 이름/URL
- **분석 소스**: 채용공고 + 특허 + 보도자료
- **출력**: "쟤네 다음에 이거 출시할 듯" 예측
- **가치**: 경쟁 전략 수립

---

### 1.4 관점 전환 스킬

#### 악마의 변호인
- **입력**: 내 아이디어/계획
- **기능**: 의도적으로 반박
- **출력**: "이거 왜 안 될까?" 10가지 이유
- **가치**: 아이디어 강화, 약점 사전 보완

#### 고객 빙의
- **입력**: 페르소나 설정 (나이, 직업, 성향 등)
- **기능**: 그 사람 관점에서 제품/서비스 사용 시뮬레이션
- **출력**: 불편한 포인트, 개선점 리스트
- **가치**: 진짜 사용자 관점 확보

#### 10년 후 회고
- **입력**: 현재 의사결정 사항
- **기능**: 미래 시점에서 현재 결정 평가
- **출력**: "2035년에서 보면 이 결정이 어떻게 보일까"
- **가치**: 장기적 관점, 후회 없는 결정

---

### 1.5 연결 스킬

#### 인맥 브릿지
- **입력**: 해결하고 싶은 문제
- **분석**: 링크드인 + 사내 인맥 분석
- **출력**: "이 문제 해결하려면 누구랑 얘기해야 해?" 추천
- **가치**: 효율적인 네트워킹

#### 지식 고고학
- **입력**: 회사 드라이브/문서 저장소
- **기능**: 전체 스캔 후 잊혀진 자료 발굴
- **출력**: "3년 전에 비슷한 거 했었는데..." 자동 발견
- **가치**: 중복 작업 방지, 과거 자산 활용

#### 우연 생성기
- **기능**: 매일 랜덤하게 "오늘의 영감" 제공
- **특징**: 전혀 관련 없어 보이는데 묘하게 연결되는 정보
- **가치**: 세렌디피티, 창의적 자극

## 2. 실용적 스킬 (User-Focused)

> 핵심 원칙: "매일 쓰는, 없으면 못 사는" 스킬

### 2.1 회의 끝나면 자동 정리 ⭐ 최우선
- **입력**: 회의 음성/텍스트
- **출력**:
  - 3줄 요약
  - 결정된 사항
  - 액션 아이템 (담당자 + 기한)
  - 슬랙/노션 자동 전송
- **타겟**: 모든 직장인 (매일 사용)
- **기대효과**: 회의 후 정리 시간 90% 단축

### 2.2 캡처 한 장으로 데이터 입력
- **입력**: 영수증/명함/계약서 사진
- **기능**:
  - 자동 필드 추출 (OCR + AI)
  - 엑셀/ERP에 바로 입력
- **타겟**: 경리팀, 영업팀
- **기대효과**: 수작업 데이터 입력 제거

### 2.3 긴 문서 → 1분 브리핑 ⭐ 진입장벽 낮음
- **입력**: PDF/보고서/이메일 스레드
- **출력**:
  - 핵심 3문장
  - 내가 해야 할 것
  - 주의할 점
- **타겟**: 모든 직장인
- **사용 시나리오**: "이거 읽어봐" 링크 받으면 바로 사용

### 2.4 못생긴 데이터 → 예쁜 차트 ⭐ 반응 확실
- **입력**: 엑셀/CSV 파일
- **출력**:
  - 자동 인사이트 발견
  - 발표용 차트 3종 생성
  - "이런 트렌드가 보입니다" 코멘트
- **타겟**: 기획자, 마케터, 경영진 보고하는 사람
- **사용 시나리오**: 주간/월간 보고서 작성

### 2.5 고객 문의 → 30초 답변
- **입력**: 고객 메시지 붙여넣기
- **출력**:
  - 문제 파악
  - 답변 초안 (우리 회사 말투로)
  - 관련 FAQ 링크 첨부
- **타겟**: CS팀, 1인 기업
- **기대효과**: 응답 시간 10배 단축

### 2.6 비즈니스 번역 (한글 ↔ 영어)
- **차별점**: 단순 번역이 아닌
  - 비즈니스 맥락 이해
  - 업계 용어 정확히 번역
  - 문화적 뉘앙스 조절
- **타겟**: 해외 거래처 있는 회사
- **사용 시나리오**: 이메일, 계약서, 제안서

## 3. 사용자가 좋아하는 패턴

| 패턴 | 설명 | 예시 |
|------|------|------|
| **즉시 결과** | 기다리기 싫음 | 붙여넣기 → 바로 결과 |
| **반복 제거** | 시간 = 돈 | 매주 하던 작업 자동화 |
| **전문가 수준** | 혼자서 못하는 것 | 디자이너급 결과물 |
| **실수 방지** | 불안함 해소 | 계약서 검토, 숫자 체크 |

## 4. 개발 우선순위 제안

### Phase 1: 즉시 가치 체감 (MVP)
1. **문서 요약** - 진입장벽 낮음, 바로 가치 체감
2. **회의록 자동 정리** - 모든 회사에서 매일 사용
3. **데이터 → 차트** - "와 이거 편하다" 반응 확실

### Phase 2: 업무 자동화
4. **고객 문의 답변**
5. **비즈니스 번역**
6. **캡처 → 데이터 입력**

### Phase 3: 차별화 스킬
7. **악마의 변호인**
8. **실패 예언자**
9. **경쟁사 다음 수 예측**

### Phase 4: 고급 스킬
10. **나 대신 답장** (학습 필요)
11. **트렌드 레이더** (데이터 소스 확보 필요)
12. **아이디어 충돌기**
